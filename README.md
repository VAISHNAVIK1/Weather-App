Weather App

A simple weather application built using HTML, CSS, and JavaScript.
This project was created to practice front-end fundamentals such as webpage structure, styling, and basic JavaScript interactivity.

1.Features

User can enter a city name in the input box
Displays weather details stored/handled in the script
Simple JavaScript logic for showing results on button click
Clean and responsive UI using CSS
Beginner-friendly project structure

2.Technologies Used

HTML — Page Structure
CSS — Styling & Layout
JavaScript — Basic functions & content updates

3.Project Structure

Weather-App/
│── index.html
│── style.css
│── script.js
└── assets/ (optional)

4.How to Run

1. Download or clone the repository
2. Open the folder
3. Double-click index.html
4. The Weather App will open in your browser
